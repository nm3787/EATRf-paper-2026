# EATR-flooding

This repository contains the data and scripts necessary to replicate the findings presented in the article "Stepping up enhanced rate calculations with EATR-flooding" http://arxiv.org/abs/2604.27136

`Data/` (will) contain all of the COLVAR and PLUMED log files used in the EATR-flooding analyses.  
`Inputs/` (will) contain all of the PLUMED scripts, as well as the LAMMPS and GROMACS files needed to run the protein G and cavity models.  
`Results/` contains all of the rates presented in the article, as well as the Jupyter notebook files used to create the figures.  
`Scripts/` (will) contain all of the batch scripts that were used to run the analyses used in the article.  
`Software/` contains the Python files needed to run the analyses. Updated versions of the files can be found at <https://github.com/nm3787/EATR-new-rate-scripts>
